## ✨ Features

*   **Pure CSS Animations**: Smooth, high-performance visual effects built entirely with CSS (`@keyframes`, transitions, and transforms) without relying on JavaScript libraries.
*   **Desktop-First Layout**: Currently optimized for large screens and desktop viewports to deliver the best visual experience.

## 🛠️ Tech Stack

*   **HTML5**: Semantic markup structuring.
*   **Pure CSS3**: Custom animations, styling, and structural layout.
